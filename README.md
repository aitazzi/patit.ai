# patit.ai

patit.ai is a tool to help clients generate their patents.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install patit.ai.

```bash
pip install -r requirements
```

## Usage

```terminal
streamlit run 1_📝_form.py
```
